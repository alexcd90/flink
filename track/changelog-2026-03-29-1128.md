## 主题趋势

1. **BITMAP 类型全栈支持**：从 core DataStream API → Table common → Table Planner，完整引入 BITMAP 数据类型，基于 RoaringBitmap 实现，覆盖序列化、类型系统、SQL 函数等各层。
2. **Application 模式能力增强**：ApplicationStore 持久化、多 Batch Job 支持、JAR 共享复用三个 commit 系列性地增强了 Application 模式的生产可用性。
3. **Materialized Table 演进**：`START_MODE` 子句、不可变列约束等特性持续完善 Materialized Table 的 DDL 表达能力。
4. **自适应调度器可观测性**：Web UI 新增弹性伸缩历史页面，提升运维体验。
